Relative and Absolute URL's

Relative URL's are used to link to resources within the same website. They are relative to the current page's URL. For example, if you are on the page "http://example.com/about.html" and you want to link to "http://example.com/contact.html", you can use a relative URL like "contact.html".

Absolute URL's, on the other hand, include the full path to the resource, including the protocol (http:// or https://) and the domain name. For example, "http://example.com/contact.html" is an absolute URL. Absolute URL's are used when linking to resources on different websites or when you want to ensure that the link works regardless of the current page's URL.
